# Хрустальный шар с огнем

Зависимости:
    cmake
    opencl-headers
    intel-opencl-icd
    ocl-icd-opencl-dev
    sudo apt-get install libassimp-dev

Нужна реализация OpenCL от вашего производителя видеокарты

Сборка и запуск:

    cmake .
    make
    cd bin
    ./raytracer
